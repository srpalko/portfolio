About Me

I am a self-taught software developer. The purpose of this website is to provide a place to showcase my work. In the projects section of the page you will find links to both the working apps and the GitHub repository containing their code. I mainly study and develop in HTML, CSS (and Sass), JavaScript, Python, Bootstrap, jQuery, Flask, React, and Redux. I also spend a good deal of time studying various software engineering topics. I am currently working on some projects that display remote data in various visual formats using SVG, D3, and JSON APIs and AJAX.
